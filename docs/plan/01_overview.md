# 项目规划总览（第一版）

## 目标
- 建立包含供应商报价、内部订单、卡派订单的综合 ERP
- 自动同步领星 FBM 订单并生成内部订单
- 支持多包裹尺寸与物流信息展示
- 支持供应商报价回传并更新内部订单
- 支持卡派字段导出

## 核心模块
- 领星订单集成
- 内部订单中心
- 供应商报价
- 卡派导出
- 权限与审计

## 核心流程
- 领星拉单 -> 内部订单生成 -> 发起报价 -> 报价回传 -> 内部确认 -> 卡派导出

## 里程碑（建议）
- M1: 领星对接与内部订单入库
- M2: 内部订单全字段与多包裹展示
- M3: 供应商报价流程
- M4: 卡派导出
- M5: 权限与审计完善
