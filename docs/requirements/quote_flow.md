# 供应商报价回传与状态流转（已确认）

## 报价流程概述
- 内部订单发起报价
- 系统生成供应商报价单模板
- 仅将可见数据导入报价单模板
- 供应商填写报价并提交
- 价格信息自动回传内部报价单

## 回传字段映射
- 报价单价 -> 内部报价单 供应商单价
- 报价总价 -> 内部报价单 供应商总价
- 交期 -> 内部报价单 交期
- 备注 -> 内部报价单 供应商备注

## 覆盖/追加规则
- 供应商价格字段覆盖内部报价单
- 交期与备注覆盖同字段
- 已确认的报价需人工确认后允许覆盖

## 状态流转
- 待报价 -> 已报价 -> 已确认
- 待报价 -> 已报价 -> 已驳回

## 异常与重试
- 回传失败自动重试 3 次
- 重复回传仅保留最新一次
